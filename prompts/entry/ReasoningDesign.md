SCOPE

--------------------------------------------------
REASONING STRUCTURE STANDARDS

Every reasoning module must include:
- Problem statement and decomposition

- Decision criteria at each branching point

- Reasoning chains must be deterministic where possible
- Ambiguous decision points must be flagged and resolved explicitly

- Reasoning must remain consistent when inputs vary within defined bounds
